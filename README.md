# nrel-pipes
The PIPES SDK for developers using NREL PIPES.

## Installation
Install this package,

```bash
$ pip install git+https://github.com/nrel-pipes/nrel-pipes@master#egg=nrel_pipes
```

## Usage
How to use in development?

```python
from pipes.core import Pipes

pipes = Pipes()
pipes.get_project()
```
